Designed For Daily Routing

熟悉的代理工作流，整理成乾淨的 Mac App

一鍵連線

首頁大按鈕搭配規則、全域、直連三種模式，平常使用不用鑽進複雜設定。

訂閱與節點管理

支援常見節點 URI、base64 訂閱、剪貼簿匯入、手動編輯與分享連結輸出。

規則分流

用網域、關鍵字、IP CIDR、GeoIP、Geosite 與程序規則決定代理、直連或拒絕。

TUN 增強模式

Developer ID 直發版可接管終端機、Docker 等不吃系統代理的流量。

Native macOS Feel

選單列常駐，視窗介面不打擾

ShadowSpace 用 SwiftUI 做出接近系統原生的操作節奏：首頁看狀態、節點頁管理伺服器、 規則頁調整分流、連線頁查看目前流量。

Distribution Routes

兩條發佈路線，各自保持清楚

Developer ID 直發版

適合需要完整協議、TUN、系統代理控制、GeoIP / Geosite 與 DNS 分流的使用者。 目前下載的 DMG 就是這條完整路線。

  • Shadowsocks、VMess、VLESS、Trojan、Hysteria2、TUIC、SOCKS5、WireGuard
  • 可用 sing-box 完整引擎與 TUN 增強模式
  • Developer ID 簽章與 Apple 公證

Mac App Store 版

為沙箱與審查相容另外準備的原生透明代理路線，不混入外部核心、 管理員 TUN 或系統代理修改流程。

  • 以 ShadowCore 與 NetworkExtension 為核心
  • 支援 Shadowsocks、Trojan、VLESS、SOCKS5 等原生路徑
  • 功能範圍比直發版保守，保持 App Store 相容性
